About the Project

About SiteSpy

SiteSpy is a mobile-first construction project records and cost estimation app built with Expo React Native, Firebase, and Firestore. It helps construction teams capture measurements, estimate material costs, and review project history on any device — even in low-connectivity field conditions.

My Contribution

My Contribution

I built the screens and reusable components that let SiteSpy users manage their personal information and app preferences. ProfileScreen.js lets users view and update their account details. SettingsScreen.js provides app-level configuration options. AppButton.js is the shared button component used throughout the app.
