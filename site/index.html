About the Project

About SiteSpy

SiteSpy is a mobile-first construction project records and cost estimation app built with Expo React Native, Firebase, and Firestore. It helps construction teams capture measurements, estimate material costs, organise site records, and review project history on any device — even in low-connectivity field conditions.

My Contribution

My Contribution

I built the screens and reusable components that let SiteSpy users manage their personal information and app preferences. ProfileScreen.js lets users view and update their account details. SettingsScreen.js provides app-level configuration options. AppButton.js is the shared button component used throughout the app, ensuring consistent interactive feedback and accessible tap targets across all screens.

Reflection

My Reflection

Shared components look simple to build but are hard to build well. Every screen that uses AppButton needs it to behave consistently regardless of context, and profile/settings screens need to handle edge cases like missing data or failed updates gracefully. I learned how to design components that are flexible enough to serve multiple use cases without breaking.
